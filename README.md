# Portable-Info-Structure
**Project title:**   
MSIM Course Recommendation and Review System

**Overview:**    
This project is a lightweight application that helps MSIM students explore courses using scraped course descriptions, sample reviews, and simple recommendation logic.

**Problem:**  
Students often struggle to choose classes for the next quarter because information is scattered across official course pages and word-of-mouth advice.  

**Data source:**  
Publicly accessible UW Information Management & Technology course description pages.  

**Features:**  
1.scrape course title, credits, and description  
2.organize course information into a structured dataset  
3.recommend courses based on interests or career goals  
4.support future addition of reviews and ratings  

**How to run:**  
open the notebook in Google Colab  
run the scraping cells  
run the recommendation function  
enter a keyword such as UX, AI, Product, or Data
